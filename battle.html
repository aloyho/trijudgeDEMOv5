Battle Judge — How to Use
Quick Guide
SCORE MODE
Paste your participant lineup using the Participant Sequence block. Names auto-fill as you navigate Next/Prev. Select dancer count (1, 2, 2v2, 3v3, 5v5). Set score scale. Tap dots for Body, Mind, Soul. Lock Score to save.
WEIGHTAGE MODE
Paste your bracket text — matches are auto-detected. Click a match to load the dancer names. Choose how many rounds. Each round has its own Body/Mind/Soul faders and scale. Lock In Verdict to save.
MULTI-ROUND
Set 2–5 rounds using the Number of Rounds control. Each round gets independent faders and its own score scale. The overall verdict aggregates scores across all rounds.
EXPORT
Tap + → Share / Export to copy or download all results.
